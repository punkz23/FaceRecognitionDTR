# UI Style Guide (Material 3)

## 1. Color Palette
The app uses a professional blue-based theme to convey trust and security.

*   **Primary Color:** `Color(0xFF0D47A1)` (Deep Blue) - Used for primary buttons, headers.
*   **Secondary Color:** `Color(0xFF42A5F5)` (Light Blue) - Accents, floating action buttons.
*   **Background:** `Color(0xFFF5F5F5)` (Light Grey) - App background.
*   **Surface:** `Color(0xFFFFFFFF)` (White) - Cards, dialogs.
*   **Error:** `Color(0xFFB00020)` (Red) - Error messages, "Clock Out" button.
*   **Success:** `Color(0xFF2E7D32)` (Green) - "Clock In" button, success status.

## 2. Typography
Font Family: **Roboto** (Android default) or **Poppins** (Modern geometric).

*   **Headline Large:** Size 32, Bold (Dashboard Greeting).
*   **Title Medium:** Size 16, Medium (Card Titles).
*   **Body Large:** Size 16, Regular (Input fields, main text).
*   **Label Small:** Size 11, Medium (Captions, timestamps).

## 3. Component Styles
### 3.1 Buttons
*   **Elevated Button:** Rounded corners (radius 12), full width, uppercase text.
*   **Outlined Button:** Border width 2, primary color text.

### 3.2 Cards
*   **Elevation:** 2 (Subtle shadow).
*   **Corner Radius:** 16.
*   **Padding:** 16 all around.

### 3.3 Inputs
*   **Style:** `OutlineInputBorder`.
*   **Active Color:** Primary Blue.
*   **Error Border:** Red.

## 4. Iconography
Use **Material Icons (Filled)** for clarity.
*   `face`: Profile/Enrollment.
*   `access_time`: History/Logs.
*   `camera_alt`: Capture action.
*   `verified_user`: Admin/Verified status.
